# HmNewVersionDownloader

## 秀丸の最新バージョンを「ダウンロード・解凍・秀丸への上書き」をして、全自動で秀丸をアップデートする。

- 整数 32bit (Beta/Release)
- 整数 64bit (Beta/Release)
- 浮動小数 32bit (Beta/Release)
- 浮動小数 64bit (Beta/Release)

を対象とする。
